# myChat

## Introduction

__myChat__ is a __Peer-to-Peer__ (__P2P__) messaging application which allows users to __exchange encrypted messages__ and __files__
directly over a network without a centralised server.

## Development Tools & Requirements

* `Linux (Ubuntu 20.04)`.
* Text editor: `ms vscode` was used. Note: any text editor will do, should you wish to change parts of the software.
* Compiler: `gcc 8 or later` for `C++17` support.
* [Boost.Asio](https://www.boost.org/doc/user-guide/task-networking.html) library for networking.
